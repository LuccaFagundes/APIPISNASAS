# NASA APOD Viewer 🚂🚂🚂

Uma aplicação web construída **React** e **Vite** que consome a API **Astronomy Picture of The Day (APOD)** da NASA. O projeto exibe diariamente uma imagem (ou vídeo) do universo, acompanhando de título, descrição e outras informações.
<img width="1919" height="882" alt="image" src="https://github.com/user-attachments/assets/0b1a80ab-8c0e-4437-9208-5ddb17af6195" />

## Tecnologias Utilizadas

- React
- Vite
- Javascript (ES6+)
- CSS
- API pública da NASA (APOD)

## Funcionalidades
- Exibição de imagens e vídeo do dia
- Título, descrição e data
- Indicador de carregamento
